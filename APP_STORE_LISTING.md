## FAQs (Add to App Listing)

### Will this work with my theme?
Yes! Infinite Scroll Pro automatically detects and works with any Shopify theme, including Dawn, Debut, Brooklyn, Prestige, Empire, and all others.

### Does it work with filters and sorting?
Absolutely. The app automatically integrates with your theme's existing filters and sorting without any configuration.

### Will it slow down my store?
No. The app uses modern, efficient APIs and adds minimal JavaScript. It's designed for maximum performance.

### Is it really free?
Yes, completely free forever. No trials, no premium plans, no hidden fees.

### Can I customize the appearance?
Yes, you can customize button colors, text, loading messages, and behavior from the Shopify theme editor.

### Does it affect SEO?
No negative impact. You can optionally enable URL updates for better SEO.

### Do I need coding skills?
No coding required. Install the app, add the block in theme editor, and you're done.

### Does it work on mobile?
Yes, fully responsive and optimized for mobile devices.

### Can I use a "Load More" button instead?
Yes, you can switch between auto-scroll and manual "Load More" button mode.

### What if I switch themes?
It will automatically work with your new theme. No reinstallation needed.
