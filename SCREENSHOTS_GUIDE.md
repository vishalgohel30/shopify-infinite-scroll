# Screenshots Guide for Shopify App Store

This guide will help you create professional screenshots for your Infinite Scroll Pro app listing.

---

## Requirements

### Shopify's Requirements

| Specification | Requirement |
|---------------|-------------|
| **Minimum Screenshots** | 3 |
| **Recommended Screenshots** | 5 |
| **Dimensions** | 1280x720px (16:9 ratio) or 1920x1080px |
| **Format** | PNG or JPG |
| **Max File Size** | 5MB per image |
| **Content** | Must show actual app functionality |

## How to Take Screenshots

### Preparation

#### 1. Set Up Demo Store

Create a professional-looking demo store:

**Store Setup:**
- Use a popular theme (Dawn, Debut, or similar)
- Add 50-100 realistic products
- Use high-quality product images
- Create organized collections
- Set up filters and sorting

**Products to Use:**
- Fashion items (universally appealing)
- Home decor (shows versatility)
- Electronics (professional look)
- Mix of prices and variants

**What to Avoid:**
- "Test Product 1, 2, 3" names
- Placeholder images
- Lorem ipsum text
- Unrealistic data

#### 2. Install Infinite Scroll

- Install your app on the demo store
- Add block to collection page
- Configure with professional settings
- Test that everything works

#### 3. Choose Screenshot Tool

**Option A: Browser Developer Tools**
```
Chrome/Edge:
1. Press F12
2. Click device toggle (Ctrl+Shift+M)
3. Select "Responsive" or specific device
4. Set to 1280x720 or 1920x1080
5. Take screenshot with built-in tool
```

**Option B: Screenshot Extensions**
- Awesome Screenshot
- Nimbus Screenshot
- FireShot

**Option C: Design Tools**
- Figma (import screenshot, add annotations)
- Canva (easy annotations and text)
- Photoshop (professional editing)

## Taking Each Screenshot

### Screenshot 1: Collection Page Hero Shot

**Steps:**
1. Navigate to a collection with 30+ products
2. Scroll down so products are loading
3. Capture the moment loading indicator is visible (or just after)
4. Ensure products, navigation, and branding visible

**Browser Setup:**
- Full screen width
- Hide bookmarks bar
- Clean URL (use custom domain if possible)
- Zoom level: 100%

**Pro Tips:**
- Time screenshot when loading spinner visible
- Show 2-3 rows of products
- Include some white space
- Ensure theme looks professional

---

### Screenshot 2: Theme Editor Settings

**Steps:**
1. Go to Online Store → Themes → Customize
2. Navigate to collection page
3. Click Infinite Scroll block to expand settings
4. Ensure all key settings visible
5. Take full screenshot of editor

**What to Include:**
- Left sidebar with all settings
- Right preview pane showing collection
- Shopify theme editor UI
- All customization options visible

**Pro Tips:**
- Expand all setting sections
- Set professional demo values
- Use readable font sizes
- Clean, uncluttered sidebar

---

## Adding Annotations

### Tools for Annotations

**Free Tools:**
- Canva (easiest, templates available)
- GIMP (open source)
- Paint.NET (Windows)
- Preview (Mac)

**Paid Tools:**
- Figma (best for professional results)
- Adobe Photoshop
- Sketch

### Annotation Guidelines

**Text Overlays:**
- Use large, readable fonts (24-36px)
- High contrast (white text on dark overlay, or vice versa)
- Short, punchy phrases
- Maximum 5-7 words per text

**Arrows:**
- Bold, obvious arrows
- Bright colors (red, blue, green)
- Point to specific features
- Not too many (1-2 per screenshot)

**Highlights:**
- Circles or boxes around key features
- Semi-transparent overlays
- Consistent color scheme
- Don't obstruct important content

**Color Scheme:**
- Use Shopify green (#5C6AC4) for consistency
- White for text on dark backgrounds
- Dark for text on light backgrounds
- Maintain brand consistency

## Quality Checklist

Before uploading screenshots, verify:

### Technical Quality
- [ ] Correct dimensions (1280x720 or 1920x1080)
- [ ] High resolution, no blurriness
- [ ] PNG format for best quality
- [ ] Under 5MB file size
- [ ] No compression artifacts

### Content Quality
- [ ] Professional demo store
- [ ] Real product names and images
- [ ] Clean, organized layout
- [ ] No test/dummy data visible
- [ ] Brand consistent styling

### Annotations Quality
- [ ] Text is readable at thumbnail size
- [ ] Arrows/highlights are obvious
- [ ] Not too cluttered
- [ ] Colors are consistent
- [ ] Professional appearance

### Functional Display
- [ ] Shows actual app functionality
- [ ] Key features highlighted
- [ ] Easy to understand at a glance
- [ ] Accurately represents the app
- [ ] No misleading information

## Screenshot Optimization

### Before Uploading

**1. Resize if Needed**
- Ensure exact dimensions
- Use high-quality resize algorithms
- Maintain aspect ratio

**2. Optimize File Size**
- Use PNG for sharp text
- Compress without losing quality
- Tools: TinyPNG, ImageOptim

**3. Test Thumbnails**
- View at small size (200x113px)
- Ensure text still readable
- Key elements visible

**4. Get Feedback**
- Show to someone unfamiliar with the app
- Ask: "What does this app do?"
- Adjust if message unclear

## Common Mistakes to Avoid

❌ **Don't:**
- Use test products or dummy data
- Include personal information
- Show admin panels (except settings)
- Use low-resolution images
- Over-annotate (too much text)
- Show errors or broken elements
- Use inconsistent styling
- Include competitor names
- Show price as only selling point

✅ **Do:**
- Use professional demo data
- Show real-world scenarios
- Highlight key benefits
- Keep annotations minimal
- Ensure high quality
- Maintain consistent branding
- Show actual functionality
- Make benefits obvious

## Final Steps

### Before Submission

1. **Review All Screenshots**
   - Open all 5 screenshots
   - View at full size
   - Check for typos in annotations
   - Verify professional quality

2. **Order Correctly**
   1. Hero shot (most impressive)
   2. Theme editor (shows ease)
   3. Mobile (shows responsiveness)
   4. Filter integration (shows compatibility)
   5. Load More button (shows flexibility)

3. **Upload to Partner Dashboard**
   - Go to App Listing
   - Upload in correct order
   - Add alt text for accessibility
   - Save and preview

4. **Test Listing Preview**
   - View how screenshots appear in listing
   - Check thumbnail quality
   - Ensure story flows well
   - Make adjustments if needed
